# AI Voice Agent Appointment Booking System

## Overview
This project is an AI-powered voice assistant that automates appointment booking for medical and service businesses.

It allows customers to call, interact with an AI agent, and get scheduled without human involvement.

## Features
- AI voice interaction
- Automated data collection
- Appointment scheduling
- Google Calendar integration
- Slack notifications
- Workflow automation

## Technology Used
- VAPI (Voice AI)
- n8n (Automation Platform)
- Google Calendar API
- Slack API
- Webhooks

## System Workflow
1. Customer calls the AI number
2. AI collects appointment details
3. n8n processes the request
4. Calendar availability is checked
5. Appointment is booked
6. Staff is notified on Slack
